# The Furry One

A small 2D dodgeball tag game prototype.

## Controls
- Move: WASD / Arrow Keys
- Aim: Mouse
- Throw (only when you are **the furry one**): click + hold to charge, release to throw

Goal: spend the least total time as **the furry one**.

## Dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
